# 👻 Ghostub

**Géocaching émotionnel** — des messages invisibles ancrés dans les lieux réels.

> *Des inconnus ont laissé quelque chose ici. Approchez-vous.*

## C'est quoi ?

Ghostub est une PWA mobile qui permet de déposer des messages géolocalisés, invisibles jusqu'à ce que quelqu'un passe physiquement à proximité.

## Fonctionnalités

- 📍 Dépôt de messages ancrés à une position GPS
- 🌫️ Radar de détection des messages proches
- 🗺️ Carte interactive des messages
- ✦ Système de résonance
- 👻 Mode anonyme ou pseudo
- ⏳ Durée de vie configurable (24h → éternel)
- 🔔 Notifications push
- 🎥 Vidéo / photo (Premium)

## Stack

- PWA vanilla HTML/CSS/JS (fichier unique `index.html`)
- Firebase Auth + Firestore
- Cloudinary (médias)
- Leaflet.js / OpenStreetMap
- Hébergé sur GitHub Pages
